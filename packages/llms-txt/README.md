# @mdream/llms-txt

Engine-agnostic [llms.txt](https://llmstxt.org) artifact generation. Consumes pre-converted markdown (`ProcessedFile[]`) and produces `llms.txt`, `llms-full.txt`, and individual markdown files.

**Zero dependency on mdream or any HTML→Markdown engine.** Bring your own converter.

## Installation

```bash
pnpm add @mdream/llms-txt
```

## Usage

### With mdream (or any converter)

```ts
import { generateLlmsTxtArtifacts, processHtmlFiles } from '@mdream/llms-txt'
import { htmlToMarkdown } from 'mdream'

// processHtmlFiles reads HTML files and delegates conversion to your callback
const files = await processHtmlFiles('dist/**/*.html', (html, url) => {
  const { markdown } = htmlToMarkdown(html, { origin: url })
  return { markdown }
}, { origin: 'https://example.com' })

const result = await generateLlmsTxtArtifacts({
  files,
  siteName: 'My Site',
  origin: 'https://example.com',
  generateFull: true,
  generateMarkdown: true,
})

console.log(result.llmsTxt) // llms.txt content
console.log(result.llmsFullTxt) // llms-full.txt content
console.log(result.markdownFiles) // [{ path, content }]
```

### Streaming

Write llms.txt incrementally without holding all content in memory:

```ts
import { createLlmsTxtStream } from '@mdream/llms-txt'

const stream = createLlmsTxtStream({
  siteName: 'My Docs',
  description: 'Documentation site',
  origin: 'https://example.com',
  outputDir: './dist',
  generateFull: true,
})

const writer = stream.getWriter()
await writer.write({
  title: 'Home',
  content: '# Welcome\n\nHome page content.',
  url: '/',
  metadata: { description: 'Welcome page' },
})
await writer.close()
```

### Custom Sections and Notes

```ts
const result = await generateLlmsTxtArtifacts({
  files,
  siteName: 'My Site',
  sections: [
    {
      title: 'Getting Started',
      description: 'Quick start guide',
      links: [
        { title: 'Installation', href: '/install', description: 'How to install' },
        { title: 'Quick Start', href: '/quickstart' },
      ],
    },
  ],
  notes: ['Generated by mdream', 'Last updated: 2024'],
})
```

### From Pre-Converted Markdown (no HTML)

```ts
import { generateLlmsTxtArtifacts } from '@mdream/llms-txt'

const result = await generateLlmsTxtArtifacts({
  files: [
    { title: 'Home', content: '# Welcome', url: '/' },
    { title: 'About', content: '# About Us', url: '/about', metadata: { description: 'About page' } },
  ],
  siteName: 'My Site',
})
```

## API

### `processHtmlFiles(patterns, converter, options?)`

Reads HTML files matching glob patterns and converts them using your callback.

- **patterns**: `string | string[]` — glob patterns for HTML files
- **converter**: `(html: string, url: string) => { markdown: string, metadata?: { title?, description?, keywords?, author? } }`
- **options.origin**: Optional origin URL

Returns `Promise<ProcessedFile[]>`.

### `generateLlmsTxtArtifacts(options)`

Generates llms.txt artifacts from processed files.

- **files**: `ProcessedFile[]` — pre-converted markdown files
- **siteName**: Site name for the header
- **description**: Site description
- **origin**: Canonical origin URL
- **generateFull**: Generate `llms-full.txt` (default: `false`)
- **generateMarkdown**: Generate individual `.md` files (default: `false`)
- **sections**: Custom sections before pages
- **notes**: Footer notes

Returns `{ llmsTxt, llmsFullTxt?, markdownFiles?, processedFiles }`.

### `createLlmsTxtStream(options)`

Creates a `WritableStream<ProcessedFile>` that writes artifacts to disk incrementally.

## llms.txt Structure

```markdown
# Site Name

> Site description

## Custom Section

- [Link Title](url): Description

## Pages

- [Page Title](url): Page description
```
